#### Usage

```
Usage: map_segment.exe [params] input_image output_image

        -b, --blur
                Blur Output Image
        -d, --double
                Double Input Image Size
        -e, --equalize
                Equalize Output Image
        -h, --help (value:true)
                Show Help Message
        -m, --manual
                Manual Filter Mode

        input_image (value:images/usa.png)
                Input Image
        output_image (value:<none>)
                Output Image

```
