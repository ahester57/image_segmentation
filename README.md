#### Usage

```
Usage: map_segment.exe [params] input_image output_image

        -b, --blur
                Blur Output Image
        -e, --equalize
                Equalize Output Image
        -h, --help (value:true)
                Show Help Message
        --hsv_plane, -p (value:2)
                HSV Plane to Use: 0 = H, 1 = S, 2 = V
        -s, --scale (value:1.f)
                Scale Input Image Size

        input_image (value:images/usa.png)
                Input Image
        output_image (value:<none>)
                Output Image

```
